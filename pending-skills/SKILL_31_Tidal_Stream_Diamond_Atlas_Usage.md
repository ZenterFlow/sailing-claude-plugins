# SKILL: Tidal Stream Diamond and Atlas Usage

## Description
Extract direction (set) and rate (drift) of tidal streams from diamonds on charts and atlas pages.

## Prerequisites
- SKILL_27 (Tide Tables)
- SKILL_30 (Time Zones)

## Learning Objectives
- Locate tidal diamonds on chart and corresponding tables
- Use tidal stream atlas pages for detailed stream data
- Determine reference port for any table (always check header)
- Identify direction using plotter on atlas pages
- Extract spring and neap rates correctly

## Procedures
**Tidal Diamonds:**
1. Find diamond letter near your position on chart
2. Locate table on chart (usually top/bottom margin)
3. Read direction (degrees True) and rates (knots)
4. Format: Direction (3 digits), Spring rate, Neap rate

**Tidal Stream Atlas:**
1. Identify reference port (Victoria) from page header
2. Determine hour offset from HW at reference port
3. Select correct page (HW-6 to HW+6)
4. Align plotter with tidal arrow to read direction
5. Rate format: NEAPS.SPRINGS (e.g., 02.13 = 0.2 kn neaps, 1.3 kn springs)

**Key Principle:**
- Always use HW at REFERENCE PORT (Victoria) regardless of vessel location
- Even if sailing near Port Fraser, use Victoria HW times

**Direction Finding:**
- Use plotter bezel aligned with page edges (North)
- Read True direction from compass rose on plotter

## Common Errors
- Using local port HW time instead of reference port
- Reading rates backwards (springs/neaps reversed on atlas)
- Not checking which port table refers to
- Using magnetic instead of True directions

## Assessment
- Extract set/drift for 5 locations at specified times
- Demonstrate atlas page selection for given HW time
- Identify reference port from 3 different tables
