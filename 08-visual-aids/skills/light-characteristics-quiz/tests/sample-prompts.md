# Sample Test Prompts for light-characteristics-quiz

## Test 1: Start Quiz
**Prompt**: "Test me on navigation light characteristics"
**Expected**: Begin 10-question quiz with multiple choice format, covering range of characteristics.

## Test 2: Explain Characteristic
**Prompt**: "What does Fl(2) 10s mean?"
**Expected**: Explain flashing light, 2 flashes in group, complete cycle every 10 seconds.

## Test 3: Quick vs Very Quick
**Prompt**: "What's the difference between Q and VQ lights?"
**Expected**: Q = 50-60 flashes/min, VQ = 100-120 flashes/min, both too fast to count by eye.

## Test 4: Occulting vs Flashing
**Prompt**: "How do I tell occulting from flashing?"
**Expected**: Occulting = mostly light (light > dark), Flashing = mostly dark (dark > light).

## Test 5: Range and Height
**Prompt**: "Light shows Fl 5s 15m 12M. What do 15m and 12M mean?"
**Expected**: 15m = height above MHWS, 12M = nominal range in nautical miles.
