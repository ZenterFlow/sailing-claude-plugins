# SKILL: Chart Corrections and Update Management

## Description
Apply and track chart corrections using Notice to Mariners, ensuring charts remain current for safe navigation.

## Prerequisites
- Chart familiarity
- Access to Notice to Mariners

## Learning Objectives
- Locate correction information on Admiralty charts
- Interpret correction date and Notice to Mariners numbers
- Update electronic charts via official sources
- Understand limitations of correction status

## Chart Correction Data
**Paper/Raster Charts:**
- Find correction date in chart title block
- Locate latest Notice to Mariners number
- Corrections applied manually with pen/ink

**Electronic Charts (ENC):**
- Updated via database updates (not individual Notices)
- Update frequency depends on service provider
- Check "chart edition" or "update status" in ECS/ECDIS

- Some older Notices may not be incorporated
- Always verify critical information with latest NM
- Electronic charts cannot be updated incrementally like paper

## Common Errors
- Assuming chart is "fully corrected" on given date
- Failing to check for outstanding Notices
- Using outdated electronic chart data
- Not verifying temporary vs permanent corrections

## Assessment
- Interpret correction block on sample chart
- Explain update process for both paper and vector charts
- Demonstrate checking chart currency before passage
