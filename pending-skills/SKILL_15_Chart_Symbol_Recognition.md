# SKILL: Nautical Chart Symbol Recognition

## Description
Identify and interpret standard nautical chart symbols, abbreviations, and markings including aids to navigation, hazards, and infrastructure.

## Prerequisites
- Basic chart familiarity
- Understanding of port/starboard conventions

## Learning Objectives
- Correctly identify 15+ standard chart symbols without reference
- Distinguish between similar symbols (e.g., different buoy types)
- Interpret symbol modifiers and supplementary information

## Key Symbols
**Navigation Aids:**
- Major light: Tall structure with powerful light
- Perch stakes: Port (red) and starboard (green) river bank markers
- Mooring buoy: Anchor symbol in circle
- AIS buoy: Tracking buoy with "AIS" label

**Infrastructure:**
- Wind farm/wind motor: Wind turbine symbol
- Marine farm: Aquaculture installation
- Pilot boarding point: "Pilot" marking
- Customs office: Building with "Customs" label

**Hazards:**
- Wreck (part exposed): Wreck symbol with superstructure showing
- RACON (Radar Transponder): Radar reflector with Morse ID

## Common Errors
- Confusing perch stake colors (port vs starboard)
- Misidentifying wreck symbols (exposed vs submerged)
- Overlooking symbol modifiers indicating light characteristics

## Assessment
Visual identification quiz: 90% accuracy on 20 random symbols from RYA Training Almanac
