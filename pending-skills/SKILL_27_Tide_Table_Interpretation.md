# SKILL: Tide Table Interpretation and Application

## Description
Extract high/low water times and heights from tide tables and adjust for Daylight Saving Time (DST) and time zones.

## Prerequisites
- SKILL_26 (Tidal Terminology)

## Learning Objectives
- Read HW/LW times from almanac tide tables
- Adjust times for DST (add 1 hour in non-shaded periods)
- Convert between UT and local time zones (±hours)
- Determine spring/neap/intermediate status from range
- Calculate tidal range accurately

## Procedures
**Reading Tide Tables:**
1. Locate correct port and date in almanac
2. Extract HW time, HW height, LW time, LW height
3. Note time zone designation (usually UT/GMT)
4. Check DST box in almanac corner

**DST Adjustments:**
- Add 1 hour in non-shaded areas when DST active
- Apply DST correction BEFORE secondary port calculations
- Remove DST when converting to UT, add back after calculations

**Time Zone Conversions:**
- Maritime zones: Letter A-Z (Z=UT), number ±hours from UT
- **Zone -1:** Subtract 1 hour to convert to UT
- **Zone +1:** Add 1 hour to convert to UT
- DST in zone: Additional ±1 hour conversion

**Spring/Neap Determination:**
1. Calculate range: HW height - LW height
2. Compare to mean ranges box on tidal curve
3. Match to MHWS/MHWN ranges to classify

## Common Errors
- Applying DST before checking if it's in effect
- Forgetting to convert back to UT for tidal stream lookups
- Applying time zone corrections twice
- Using wrong date or port table

## Assessment
- Extract and adjust 5 tide predictions (DST + time zones) in &lt;5 min
- Correctly identify spring/neap status for 3 dates
- Explain DST application logic for secondary ports
