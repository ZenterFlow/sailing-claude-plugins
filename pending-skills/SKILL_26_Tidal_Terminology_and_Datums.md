# SKILL: Tidal Terminology and Vertical Datums

## Description
Master tidal terminology and work with chart datum, HAT, LAT, and tidal ranges for safe depth calculations.

## Prerequisites
- SKILL_25 (Tidal Theory)

## Learning Objectives
- Define MHWS, MLWS, MHWN, MLWN, HAT, LAT
- Explain chart datum as reference plane for depths
- Calculate tidal range (HW-LW)
- Apply HAT for bridge clearance calculations
- Interpret mean ranges box on tidal curves

## Key Terminology
**Vertical Reference Planes:**
- **Chart Datum (CD):** Reference plane for charted depths (near LAT)
- **HAT (Highest Astronomical Tide):** Maximum possible tide under astronomical conditions
- **LAT (Lowest Astronomical Tide):** Minimum possible tide

**Tidal Levels:**
- **MHWS:** Mean High Water Springs (average spring high)
- **MLWS:** Mean Low Water Springs (average spring low)
- **MHWN:** Mean High Water Neaps (average neap high)
- **MLWN:** Mean Low Water Neaps (average neap low)

**Practical Applications:**
- Bridge clearances USUALLY given above HAT (verify on chart)
- Light elevations above MHWS
- Drying heights = height above CD exposed at low tide
- **Range** = HW height - LW height

**Chart Interpretation:**
- Charted depth: Depth below CD (always water)
- Height of tide: Water level above CD at specific time
- Total depth = Charted depth + Height of tide

## Common Errors
- Confusing chart datum with sea level
- Using MHWS instead of HAT for clearance
- Forgetting older charts may use MHWS for clearances
- Misidentifying drying height vs charted depth
